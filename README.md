# Vibenotes Website

This is the landing page for Vibenotes, designed with an AI-first aesthetic.

## How to View
Simply open `index.html` in any modern web browser.

## Structure
- `index.html`: Main structure and content.
- `style.css`: Styling, animations, and dark mode theme (based on AppColors.swift).
- `script.js`: Scroll animations and interactivity.

## Features
- **AI Aesthetic**: Dark mode, glowing gradients, glassmorphism.
- **Animations**: Scroll-triggered fade-ins, floating blobs.
- **Content**: Structured according to the provided pitch script.
